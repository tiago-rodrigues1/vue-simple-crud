# Vue Simple Crud

A very simple crud implementation using VueJS and Express

## How To Install
1. Clone the repo
2. Go to the project directory and run the commands
    1. `cd backend`
    2. `npm install`
    3. `node index.js`
    4. `cd ../frontend`
    5. `npm install`
    6. `npm run dev`
3. You are ready
